## üöÄ Getting Started

### Prerequisites

- Git installed on your machine
- A modern web browser
- (Optional) Node.js if using build tools

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/UjjavalParmar/portfolio-site.git
   cd portfolio-site
   ```

2. **Open the project:**
   ```bash
   # If using a simple setup, open index.html in your browser
   open index.html
   
   # Or if using a local server (Python):
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

3. **Customize the content:**
   - Update `index.html` with your personal information
   - Modify `styles.css` to match your brand colors
   - Add your projects and skills
   - Update contact information
